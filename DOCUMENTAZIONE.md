# Tappa n.1
## Obiettivo
L'obiettivo in questa tappa del progetto "Pac-Man" è quello di creare un sistema di layout della mappa che permetta di visualizzare il labirinto di gioco. Il layout sarà definito in un file e verrà caricato all'avvio del gioco. In seguito, si prevede di implementare un sistema per creare livelli personalizzati.
## Ostacoli
L'ostacolo principale per questa tappa è la definizione del sistema con cui salvare/caricare il layout della mappa.
## Soluzione
Inizialmente si salvava il layout in un file di testo avendo ogni tile associata a un numero che ne rappresenta il tipo (es. 0->tile con il pac-dot, 1->tile con il muro, 2->tile con il frutto, ecc.).
